# ToDo-API с матрицей Эйзенхауэра

Простой и функциональный REST API для управления задачами с классификацией по матрице Эйзенхауэра. Реализован на **FastAPI** с использованием **Pydantic** для валидации и **Swagger UI** для автоматической документации.

---

## Возможности API

✅ **CRUD-операции** над задачами  
✅ **Классификация задач** по квадрантам Эйзенхауэра (Q1–Q4)  
✅ **Валидация данных** через Pydantic-схемы  
✅ **Асинхронная обработка** запросов  
✅ **Автодокументация** Swagger UI и ReDoc  
✅ **Фильтрация и поиск** по статусу, квадранту, ключевым словам  
✅ **Статистика** по задачам  

---

## Технологии

- **Python 3.11+**
- **FastAPI** – современный веб-фреймворк
- **Uvicorn** – ASGI-сервер
- **Pydantic** – валидация данных
- **Swagger UI / ReDoc** – интерактивная документация
